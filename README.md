# Dot.Tech

#### Dot Tech is annually held intra-college tech fest conducted by IT & CS department of D.G Ruparel College.
> #### D.G Ruparel, is a premier institution in the field of higher education. The College has completed more than sixty glorious years of educating young minds.

### About Dot Tech

It's Technological Festival, an event celebrated since 2001 is organized by The IT and CS Department. 
Dot Tech is a Platform to showcase programming skills, creativity, through various activities. 
It is one of the most awaited Event in D.G Ruparel, attended by more then 2000 students, spanning across different streams and branches.
Dot Tech holds over 14 Event, spanning across 3 days.


The App is the gateway pass to the Fest, it provides a hassle-free platform to Participate in Events. 
Register once and participate in any event on fly. 

### Features:
1) Seamless, Interactive, aesthetic look and feel, Material Design based User Interface(UI), Designed with focus on User Experience(UX). 
2) Browse through Events, which have most recent and latest data.
3) Realtime updates, Everything is updated in realtime. Any change is available across all devices in an instant.
4) Support Individual and Team Participation.
5) Notification, stay up to date with the event dates, data and more all the time. Supporting Global, User, Event level channels.
